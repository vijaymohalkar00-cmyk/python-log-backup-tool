# Python Log Analyzer & Backup Tool

This project is a simple Python automation tool that:
- Analyzes log files and counts errors and warnings
- Takes backup of folders into ZIP files
- Runs from command line using a menu

## Project Files
- main.py : Main menu program
- log_analyzer.py : Log analysis module
- backup.py : Backup module
- sample_logs/system.log : Sample log file

## How to Run
1. Open terminal in project folder
2. Run:
   python main.py
3. Choose:
   - 1 to analyze log file
   - 2 to take folder backup

## Use Case
Useful for Linux Support, System Administrator, and Automation tasks.
